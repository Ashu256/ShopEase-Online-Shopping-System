# 🛒 ShopEase Online Shopping System

A web-based online shopping system developed using **Java Servlets, HTML, and CSS**.  
This project demonstrates the basic workflow of an e-commerce platform including login, product browsing, cart management, and order placement.

---

## 🚀 Features

- 🔐 User Login Authentication  
- 🛍️ Product Catalog Display  
- ➕ Add to Cart Functionality  
- 🛒 View Cart  
- 📦 Order Placement  
- 💾 Session Management using HttpSession
